# Agro Forte - Projeto Agrinho 2026

Bem-vindo ao projeto **Agro Forte**! Um site moderno e interativo sobre sustentabilidade agrícola em Itambaracá.

## 📋 Descrição

Este projeto foi desenvolvido para o **Programa Agrinho 2026** com o tema "Agro Forte". O site apresenta informações sobre a agricultura em **Itambaracá**, com foco em sustentabilidade e boas práticas agrícolas.

## 🎯 Características Principais

✅ **Design Moderno e Responsivo** - Interface limpa e verde
✅ **Dark Mode** - Botão para alternar entre modo claro e escuro
✅ **Aumento de Fonte** - Botão A+ para aumentar o tamanho da letra
✅ **Quiz de Sustentabilidade** - 8 perguntas em nível fácil
✅ **Animações Suaves** - Efeitos visuais e interatividade
✅ **Seções Informativas** - Cards explicativos sobre agro e sustentabilidade
✅ **Créditos** - Informações do aluno e professor orientador

## 📁 Estrutura do Projeto

```
├── index.html      # Arquivo HTML principal
```

## 🎨 Tema Visual

- **Cores Principais**: Verde em várias tonalidades (#2d8f3d, #52b788, #74c69d)
- **Acento**: Amarelo (#ffd60a)
- **Dark Mode**: Suporte completo
- **Animações**: Folhas flutuantes, transições suaves e efeitos de hover

## 📚 Seções do Site

1. **Início** - Hero section com apresentação impactante
2. **Sobre Itambaracá** - Informações sobre o município e suas tradições agrícolas
3. **Informações** - 6 cards com tópicos sobre:
   - Plantações Principais
   - Sustentabilidade
   - Agricultura Familiar
   - Tecnologia Agrícola
   - Conservação de Água
   - Biodiversidade

4. **Quiz** - 8 perguntas sobre sustentabilidade (nível fácil)
5. **Créditos** - Informações do projeto:
   - **Aluno**: Arthur Marques Morgante
   - **Professor Orientador**: Guilherme Cordeiro
   - **Tema**: Agro Forte
   - **Localidade**: Itambaracá - Paraná

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Navegue pelas seções usando o menu
3. Use os botões de controle:
   - 🌙 **Dark Mode** - Ativa/desativa o modo escuro
   - **A+** - Aumenta o tamanho da fonte (3 níveis)
4. Complete o quiz sobre sustentabilidade

## ⚙️ Funcionalidades JavaScript

- **Theme Toggle** - Alterna entre modo claro e escuro (salva em localStorage)
- **Font Size Toggle** - Aumenta tamanho da fonte em 3 níveis (salva em localStorage)
- **Quiz Interativo** - Validação de respostas e cálculo de pontuação
- **Smooth Scroll** - Navegação suave entre seções
- **Scroll Animations** - Animações ao entrar na visualização
